English

An Android app cloning and runtime privacy-protection solution built on a user-space sandbox architecture. No root required — run multiple isolated app instances in parallel, each with independent storage, process, and account context.

Compatibility
Production-validated support for apps protected by major packers including 360, iJiami, Bangcle, and Tencent, plus standard Google Play distribution. Adapted for DingTalk, WeCom (Enterprise WeChat), and platforms such as WhatsApp, Telegram, Facebook, Instagram, LINE, TikTok, Slack, and Zoom.
Korean locale is supported, covering Korea-market and globally distributed Korean apps; game scenarios include popular titles such as Lineage M, Black Desert Mobile, Odin: Valhalla Rising, MapleStory M, Summoners War, and Epic Seven.

Commercialization
Includes a mature license verification platform with standardized REST APIs (activation, heartbeat, remote config, announcements, etc.) and a ready-to-integrate client SDK — operation-ready upon deployment. Supports feature tailoring, white-label branding, and policy customization.

Runtime capabilities
Virtual location, device fingerprint spoofing, WiFi identity simulation, and photo/camera data substitution — full config-delivery and runtime hook pipeline. Custom development for game-related scenarios is also available.
